# Overview

The project explores an AI-assisted hiring workflow: candidates submit information, recruiters evaluate fit, and scoring logic supports structured review.

## Problem Space

Hiring systems need to reduce manual review effort without hiding bias, context, or decision criteria. AI support is useful only when it stays explainable and auditable.

## Primary Users

- Candidates submitting profiles and answers
- Recruiters reviewing candidate fit
- Maintainers managing platform behavior

## Scope

- Role-based hiring workflows
- Candidate evaluation and scoring logic
- Django application structure

## Outcomes To Evaluate

- Evaluation workflow is explainable
- Role boundaries are clear
- Risks around AI-assisted hiring are visible
